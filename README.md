# Manufacturing Inventory Performance Dashboard

## Overview
Developed an interactive Power BI dashboard using manufacturing inventory data to monitor stock levels, inventory value, material consumption, and vendor performance.

## Tools Used
- Microsoft Excel
- Power BI

## Key Features
- Inventory Value Tracking
- Stock Availability Monitoring
- Vendor Performance Analysis
- Warehouse Wise Inventory Analysis
- Top Consumed Materials
- Project Wise Consumption Analysis

## KPIs
- Total Inventory Value
- Total Stock Quantity
- Total Consumption
- Total Received Quantity
- Pending Quantity
- Low Stock Count
